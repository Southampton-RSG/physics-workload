Load equation for module co-ordinators:
  • Module CATS $C$.
  • Number of students $N_s$.
  • Number of lectures $N_l$, problem classes $N_p$, synoptic lectures $N_{ls}$, coursework $N_{cw}$.
  • Fraction of marks from coursework $M_{cw}$, exams $M_e$.
  • Fraction of marking done by co-ordinator of coursework $F_{cw}$, exams $F_e$.
  • Fixed module-specific adjustment $A$
$$ \left[ L_{lec} * (N_l + N_p + N_{ls}) \right] + \\ \left[ L_{cw, \mathrm{prep}} * N_{cw} \right] \\ + \left[ L_{cw, \mathrm{cats}} * M_{cw} * C \right] + \\ \left[ L_{cw, \mathrm{mark}} * N_{cw} * N_s * F_{cw} * \left( N_{cw} + M_{cw} * C \right) \right] + \\ \left[ L_{e, \mathrm{cats}} * M_e * C \right] + \\ \left[ L_{e, \mathrm{mark}} * N_s * F_e \right] + A $$
